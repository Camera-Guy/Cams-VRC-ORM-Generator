# Cams-VRC-ORM-Generator
Takes Metallic (Roughness Alpha) and Occlusion as usally set up for Standard or Moblile Standard Lite and combines them perfectly without any change to the shaded material

You can also use a B&W roughness texture (like what you would put in Autodesk shader) or a Albedo (or any texture with an alpha you wish to copy) and it will just work

you just put the textures and it automaticly switches modes
